\subsubsection{Rancangan Detail Komponen GUI}

Komponen GUI akan digunakan sebagai salah satu antarmuka pengguna untuk berinteraksi dengan sistem. 

% Komponen ini akan dibangun menggunakan \textit{framework} Streamlit \parencite{streamlit}, yang merupakan \textit{framework} berbasis Python untuk membangun aplikasi web interaktif. Komponen GUI akan memiliki dua fitur utama, yaitu pencarian Smart Contracts dan pengambilan data address dari Smart Contracts. Pengguna dapat memasukkan query dalam bentuk bahasa alami untuk mencari Smart Contracts yang relevan, dan juga dapat memasukkan dgraph ID untuk mendapatkan address dari Smart Contract yang dicari.
